#### JavaScriptの例

```javascript
// 森林ボランティア活動の参加者管理
class VolunteerManager {
  constructor() {
    this.volunteers = new Map();
    this.activities = [];
  }

  addVolunteer(name, email, skills = []) {
    const volunteer = {
      id: Date.now(),
      name,
      email,
      skills,
      joinDate: new Date(),
      activities: []
    };
    
    this.volunteers.set(volunteer.id, volunteer);
    return volunteer.id;
  }

  assignToActivity(volunteerId, activityName) {
    const volunteer = this.volunteers.get(volunteerId);
    if (volunteer) {
      volunteer.activities.push({
        name: activityName,
        date: new Date(),
        status: 'assigned'
      });
    }
  }

  getVolunteersBySkill(skill) {
    return Array.from(this.volunteers.values())
      .filter(v => v.skills.includes(skill));
  }
}

// 使用例
const manager = new VolunteerManager();
manager.addVolunteer('田中太郎', 'tanaka@example.com', ['植樹', '下草刈り']);
```

#### TypeScriptの例

```typescript
interface ForestActivity {
  id: string;
  name: string;
  date: Date;
  location: string;
  maxParticipants: number;
  currentParticipants: number;
  skills: string[];
  description: string;
}

interface Volunteer {
  id: string;
  name: string;
  email: string;
  phone?: string;
  skills: string[];
  joinDate: Date;
  activities: string[];
}

class ForestActivityManager {
  private activities: Map<string, ForestActivity> = new Map();
  private volunteers: Map<string, Volunteer> = new Map();

  createActivity(activity: Omit<ForestActivity, 'id' | 'currentParticipants'>): string {
    const id = crypto.randomUUID();
    const newActivity: ForestActivity = {
      ...activity,
      id,
      currentParticipants: 0
    };
    
    this.activities.set(id, newActivity);
    return id;
  }

  registerVolunteer(volunteerId: string, activityId: string): boolean {
    const activity = this.activities.get(activityId);
    const volunteer = this.volunteers.get(volunteerId);
    
    if (!activity || !volunteer) return false;
    
    if (activity.currentParticipants >= activity.maxParticipants) {
      throw new Error('活動の定員に達しています');
    }
    
    activity.currentParticipants++;
    volunteer.activities.push(activityId);
    return true;
  }
}
```

## まとめ

content collectionsを使用することで、以下のメリットがあります：

1. **型安全性**: フロントマターのスキーマ検証
2. **開発体験**: 自動補完とエラー検出
3. **パフォーマンス**: ビルド時の最適化
4. **保守性**: 構造化されたコンテンツ管理

このサンプルページを参考に、様々なコンテンツを作成してください。
